# 🚀 TrashBot – App Driven Robotic Dustbin  

♻️ **Smart Waste Segregation using AI + IoT + Robotics**  

---

## 📌 Overview  

**TrashBot** is an intelligent waste management system that automates waste segregation using **AI-based classification, robotic automation, and IoT integration**.  

It reduces manual effort, improves accuracy, and encourages responsible waste disposal through a **reward-based system**.

---

## 🎯 Key Features  

- ✨ Automated Waste Segregation  
- 🤖 Robotic Arm-based Sorting  
- 🧠 AI Classification (CLIP Model)  
- 📱 Android App for Monitoring  
- 🌐 Backend with Spring Boot  
- 📊 Real-time Waste Analytics  
- 🎁 Reward & Eco Score System  

---

## 🧠 How It Works  

flowchart TD
- A[User Drops Waste] --> B[Slider 1]
- B --> C[Sponge Rollers]
- C --> D[Slider 2]
- D --> E[Camera Capture]
- E --> F[AI Classification]
- F --> G[Robotic Arm Sorting]
- G --> H[Waste Segregated into Bin]

## 🏗️ System Architecture  

## 🔹 Hardware Layer  
- Raspberry Pi (Controller)  
- Servo Motors (Robotic Arm)  
- Gear Motors (Sliders)  
- Ultrasonic Sensors  
- Camera Module  

## 🔹 Software Layer  
- **Backend:** Spring Boot  
- **Database:** PostgreSQL  
- **Mobile App:** Android (Kotlin + XML)  
- **AI:** CLIP Model + OpenCV + TensorFlow
